Bachelor of Science in Information Technology

The Bachelor of Science in Information Technology (BSIT) program is designed to equip students with the theoretical knowledge and practical skills necessary for careers in the IT industry. Students will learn about IT infrastructure, networking, programming, database management, and cybersecurity, preparing them to solve real-world technology problems.

Core Subjects:

  • Programming Fundamentals (Java, Python, C++)
  • Database Management Systems
  • Networking and Telecommunications
  • Cybersecurity Principles
  • Web and Mobile Development
  • Software Engineering
  • Artificial Intelligence and Machine Learning

Career Opportunities:

  • Software Developer
  • Network Administrator
  • IT Support Specialist
  • Cybersecurity Analyst
  • Web Developer
  • Data Analyst
  • System Administrator

Program Highlights:

The BSIT program emphasizes hands-on training, with students participating in projects, internships, and industry-related certifications. The curriculum is constantly updated to reflect the latest trends and technologies in the field.
